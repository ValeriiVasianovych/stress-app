Fibonacci EKS Cluster Scaling Test

This application calculates the Fibonacci sequence in real time. It is used to test the EKS Kubernetes cluster scaling capabilities. Also it collects metrics in Prometheus and displays them using Grafana.

    The purpose of this application is:
  • 1. Test the CPU load of the EKS Kubernetes cluster.
  • 2. Show cluster scaling using HPA and Cluster Autoscaler.
  • 3. Collect metrics in Prometheus and display them in Grafana.

To start the calculation, send a GET request to /api/cpu?index=44
or use curl:
curl "localhost:3000/api/cpu?index=44"
